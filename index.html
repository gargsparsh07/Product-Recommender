Beauty Product Recommender System

Discover personalized beauty products using machine learning

About This Project

This recommender system was trained using the Amazon Beauty Product ratings dataset. It predicts user preferences and recommends products based on similar user behavior using matrix factorization and SVD.

User Input: User ID 10284

Recommended Products:

  • L'Oreal Paris Lash Paradise Mascara
